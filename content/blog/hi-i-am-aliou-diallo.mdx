Laravel is a powerful PHP framework that makes building web applications enjoyable and productive.
In this guide, we'll walk through setting up your first Laravel project.

## Prerequisites

- PHP 8.1 or higher
- Composer
- Node.js and npm

## Installation

First, let's create a new Laravel project using Composer:

```sh
composer create-project laravel/laravel example-app
```

```php
<?php

public function index()
{
    return view('welcome');
}
```

```html
<!DOCTYPE html>
<html lang="en">
```

## Understanding the Structure

Laravel follows a clear directory structure:

- `app/` - Contains your core application code
- `config/` - All configuration files
- `routes/` - Application routes
- `resources/` - Views, raw assets, etc.
